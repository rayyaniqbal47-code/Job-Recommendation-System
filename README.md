# Job Recommendation System (Job Portal with Content-Based Filtering)

## Description
This is a Job Recommendation System built as a job portal that uses content-based filtering to recommend relevant jobs to users based on their skills and experience.

The system helps job seekers discover suitable job opportunities by matching their profile information with job descriptions.

## Features

### User Features
- User registration and login system
- Job seeker profile creation (skills and experience)
- View and apply for jobs
- Content-based job recommendation system
- Personalized job suggestions based on profile matching
- Job search and filtering options

### Admin Features
- Admin dashboard for managing the platform
- Add, update, and delete job postings
- Manage registered users (job seekers)
- Approve or remove job listings
- Monitor job applications and platform activity

## Email Notification System
- Sends email notifications to users when a new job is added that matches their profile.
- Job recommendations are generated using content-based filtering based on user skills and experience.
- Users receive alerts for relevant job opportunities in real time, improving job discovery and engagement.

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript

## Core Concept
- Content-Based Filtering:
  The system recommends jobs by comparing user skills and experience with job descriptions and finding the best matches.
